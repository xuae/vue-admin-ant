# vue-admin-ant
> 技术栈：Vue 全家桶，Typescript，Ant Design UI组件库

## 项目规范

### 引入新模块，必须在使用的地方注释
``` javascript
/**
 * xxx
 *
 * @dependency: xxx (npm | yarn 引入的依赖包名)
 * @dependency: xxx
 */
```

### 代码有待改进，因各种原因未改需注释
``` javascript
/**
 * xxx
 *
 * @TODO: xxx
 */
```
